🎮 遊び方（10 Game - DecaMerge / Squares + Ops）

  1. 画面上で指を左右にスライドして狙いを合わせ、指を離すと投入。
  2. 1〜9の小タイル：隣り合うグループで合計=10なら即「10」に合体（2〜6個、複数同時OK）。
  3. かけ算：2 と 5が隣り合うと10に合体。
  4. ひき算（−トークン）：小タイルに触れるとその値を10−xに変換（補数）。
  5. わり算（÷トークン）：10,20,40…に触れると半分に分割（10→5+5、20→10+10…）。
  6. 同じ大タイルは軽く重なったら倍々合体。赤線より上に一定時間あるとゲームオーバー。

ヒントONで各小タイルに「10まであといくつ」を表示。
